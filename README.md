# Miliarderzy
To podróbka znanego polskiego teleturnieju "Milionerzy", w stylu faktycznej i interaktywnej gry na stronie internetowej.

## Wykorzystane języki programowania
- JavaScript (JS)
- Cascade Style Sheets (CSS)
- HyperText Markup Language (HTML)
- Python (wkrótce, jako gra w pliku .exe!)

# ⚠️ Gra nie jest jeszcze grywalna!
Proszę nie korzystać jeszcze ze strony, To dopiero testy!
Za kilka dni opublikowana będzie wersja finalna!
Informacje o wersji można znaleźć [tutaj.](#changelog)
